# podcast-transcriber

A python library for transcribing podcasts and generating a downloadable txt file from the transcription. Now only supports Spotify podcasts.

## Project structure

- `src/podcast_transcriber/spotify.py` - Spotify transcription interface skeleton
- `src/podcast_transcriber/transcript.py` - transcript export helper for `.txt` files
- `src/podcast_transcriber/__init__.py` - public package exports
